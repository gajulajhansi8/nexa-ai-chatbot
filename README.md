# 🤖 Nexa AI Chatbot

A free ChatGPT-like AI chatbot running locally on your computer.

## ✨ Features

- ✅ **100% Free** - No API costs or subscriptions
- ✅ **Private** - All data stays on your computer
- ✅ **Powered by Llama 3** - Meta's open-source model
- ✅ **Built with Streamlit** - Easy to use web interface
- ✅ **No Cloud Required** - Runs entirely locally
